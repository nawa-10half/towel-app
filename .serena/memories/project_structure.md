## Firestore Data Structure

```
/users/{userId}                        # ユーザードキュメント (groupId?)
  └── /towels/{towelId}                # タオル (ソロモード)
      ├── /records/{recordId}          # 交換記録
      └── /conditionChecks/{checkId}   # 状態診断結果

/groups/{groupId}                      # 家族グループ
  ├── /members/{userId}                # グループメンバー
  └── /towels/{towelId}               # タオル (グループモード)
      ├── /records/{recordId}
      └── /conditionChecks/{checkId}
```

Storage path: `users/{userId}/towels/{towelId}/conditions/{checkId}.jpg`
(グループ時: `groups/{groupId}/towels/{towelId}/conditions/{checkId}.jpg`)
