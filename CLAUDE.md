## Project Overview

かえたお (Kaetao) - タオルの交換タイミングを管理するiOSアプリ。Firebase (Auth/Firestore/Storage) でデータ管理。

## Tech Stack

- **UI**: SwiftUI (iOS 17+)
- **Auth**: Firebase Auth (Apple Sign-In + Google Sign-In)
- **DB**: Cloud Firestore (リアルタイムリスナー)
- **Storage**: Firebase Storage (写真アップロード)
- **State**: `@Observable` (Observation framework)
- **Notifications**: UserNotifications framework
- **Architecture**: MVVM with `@Observable` ViewModels + Singleton Services
- **Tools**: XcodeGen (project generation)
- **Dependencies**: firebase-ios-sdk (SPM), GoogleSignIn-iOS (SPM)

## Development Commands

```bash
# Generate Xcode project from project.yml
xcodegen generate

# Build via Xcode (open TowelApp.xcodeproj)
```
