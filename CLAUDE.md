## Project Overview

かえたお！ (Towel App) - タオルの交換タイミングを管理するiOSアプリ。家族間でCloudKit経由のデータ共有が可能。

## Tech Stack

- **UI**: SwiftUI (iOS 17+)
- **Persistence**: SwiftData
- **Sync/Sharing**: CloudKit (iCloud sharing across different Apple IDs)
- **Notifications**: UserNotifications framework
- **Architecture**: MVVM with `@Observable` ViewModels
- **Tools**: XcodeBuildMCP (build/test/simulator), XcodeGen (project generation)

## Development Commands

## Architecture

- **MVVM**: Views observe ViewModels via `@Observable` macro
- **SwiftData**: `@Model` classes with `@Relationship` for Towel ↔ ExchangeRecord
- **CloudKit**: `ModelConfiguration(cloudKitDatabase: .automatic)` for private + shared DB
- **Bundle Identifier**: `com.kaetao-app.TowelApp`
- **CloudKit Container**: `iCloud.com.kaetao-app.TowelApp`
