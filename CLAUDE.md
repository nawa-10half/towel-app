This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Project Overview

かえたお (Kaetao) - タオルの交換タイミングを管理するiOSアプリ。Firebase (Auth/Firestore/Storage) でデータ管理。家族グループ共有機能あり（招待コード方式）。

## Tech Stack

- **UI**: SwiftUI (iOS 17+)
- **Auth**: Firebase Auth (Apple Sign-In + Google Sign-In)
- **DB**: Cloud Firestore (リアルタイムリスナー)
- **Storage**: Firebase Storage (写真アップロード)
- **State**: `@Observable` (Observation framework)
- **Notifications**: UserNotifications framework
- **Architecture**: MVVM with `@Observable` ViewModels + Singleton Services
- **Tools**: XcodeGen (project generation)
- **Dependencies**: firebase-ios-sdk (SPM), GoogleSignIn-iOS (SPM)

## Development Commands

```bash
# Generate Xcode project from project.yml
xcodegen generate

# Build via Xcode (open TowelApp.xcodeproj)

# Firebase CLI (towel-app ルートで実行)
firebase deploy --only firestore:rules   # Firestoreルールデプロイ
firebase deploy --only storage           # Storageルールデプロイ
firebase deploy --only hosting           # Hosting (alexa-auth.html等) デプロイ

# Lambda デプロイ
cd lambda/<function-name> && zip -r ../<name>.zip . && \
aws lambda update-function-code --function-name <name> --zip-file fileb://../<name>.zip
```

## Firestore Data Structure

```
/users/{userId}                    # ユーザードキュメント (groupId?)
  └── /towels/{towelId}            # タオル (ソロモード)
      ├── /records/{recordId}      # 交換記録
      └── /conditionChecks/{checkId} # 状態診断結果

/groups/{groupId}                  # 家族グループ
  ├── /members/{userId}            # グループメンバー
  └── /towels/{towelId}           # タオル (グループモード)
      ├── /records/{recordId}
      └── /conditionChecks/{checkId}
```

Storage path: `users/{userId}/towels/{towelId}/conditions/{checkId}.jpg`
(グループ時: `groups/{groupId}/towels/{towelId}/conditions/{checkId}.jpg`)
