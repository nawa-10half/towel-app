## テスト完了 🎉

**テスト実施日**: 2026-02-22
**結果**: 全項目 OK

### テスト中に発見・修正したバグ
1. コード無効エラー後、別画面に遷移してもエラーメッセージが残る → SignInView の `.onChange(of: step)` でクリア
2. オフライン時にタオル追加・編集・交換記録がハング → Firestore 書き込みを非async（ローカルキャッシュ即時書き込み）に変更
3. Alexa から交換記録時に `lastExchangedAt` が更新されない → Lambda の `recordExchange` をバッチ書き込みに修正
4. アカウント削除後に旧コードで再ログインできる → Lambda で `newUser:true` フラグがない場合は存在しないコードを拒否
