# かえたお App Store 提出前テストチェックリスト

## テスト環境

- **実機**: iPhone（iOS 17+）
- **ネットワーク**: Wi-Fi + モバイルデータ + 機内モード
- **アカウント**: 新規 + 既存の両方
- **グループ**: ソロモード + グループモード（可能なら2台で）

---

## 1. 初回起動・オンボーディング

- [ ] アプリ削除→再インストール後、オンボーディング（4ページ）が表示される
- [ ] 各ページのスワイプ遷移が正常
- [ ] 最終ページの「はじめる」タップ → サインイン画面に遷移
- [ ] 2回目以降の起動ではオンボーディングが表示されない

## 2. 認証（リストアコード）

### 新規ユーザー
- [ ] リストアコード（XXXX-XXXX-XXXX）が自動生成される
- [ ] 「コードをコピー」→ クリップボードにコピーされる
- [ ] 「メモしました → 次へ」→ 表示名入力画面に遷移
- [ ] 表示名を入力して「はじめる」→ サインイン成功 → メイン画面へ
- [ ] 表示名が空の場合「はじめる」ボタンが無効

### 既存ユーザー（コード復元）
- [ ] 「コードをお持ちの方はこちら」→ コード入力画面に遷移
- [ ] 正しいコードを入力 → サインイン成功
- [ ] 不正なコードを入力 → エラーメッセージ表示
- [ ] 空の入力で「サインイン」ボタンが無効

### 自動サインイン
- [ ] サインイン済み状態でアプリ再起動 → 自動的にメイン画面表示
- [ ] Keychain にコードが保存されている状態で起動 → 自動サインイン

## 3. タオル管理（CRUD）

### 追加
- [ ] 「+」ボタン → タオル追加フォーム表示
- [ ] 名前・場所を入力（必須）、アイコン選択、交換間隔設定（1〜30日）
- [ ] 場所クイック選択ボタン（浴室、キッチン等7種）が動作する
- [ ] アイコン12種の選択が動作する
- [ ] 「追加」→ 一覧に即座に反映
- [ ] 名前または場所が空の場合「追加」ボタンが無効

### 編集
- [ ] 詳細画面 →「編集」→ 編集フォーム表示（既存値がプリセット）
- [ ] 各フィールドを変更して「保存」→ 反映される

### 削除
- [ ] 一覧で右スワイプ →「削除」→ タオル削除
- [ ] 関連するサブコレクション（records, conditionChecks）も削除される

### 一覧表示
- [ ] ステータス順にソート（交換時期超過 → もうすぐ → 余裕あり）
- [ ] 検索バーで名前・場所でフィルタリングできる
- [ ] 経過日数・ステータスバッジが正しく表示される
- [ ] タオル0件時の空状態表示（「タオルがありません」）

## 4. 交換記録

- [ ] 一覧で左スワイプ →「交換した！」→ 記録シート表示
- [ ] 詳細画面の「交換した！」ボタン → 記録シート表示
- [ ] 日時選択（過去のみ）+ メモ（任意）入力
- [ ] 「記録する」→ 一覧の経過日数が即座に更新される
- [ ] 「記録する」→ 詳細画面の経過日数・最終交換日・次回交換日が更新される
- [ ] 詳細画面の交換履歴セクションに記録が表示される
- [ ] 交換履歴のスワイプ削除が動作する
- [ ] 記録削除後、経過日数が再計算される（前の記録の日付 or createdAt）
- [ ] 全記録削除後、経過日数が createdAt ベースに戻る

## 5. AI 状態診断

### カメラ撮影
- [ ] 「カメラで撮影」→ カメラ権限ダイアログ表示（初回）
- [ ] 権限許可 → カメラ起動
- [ ] 権限拒否 → 「設定を開く」アラート表示
- [ ] 写真撮影 → 診断開始（「診断中...」表示）
- [ ] 診断完了 → 結果（スコア・コメント・推奨）が表示される

### フォトライブラリ
- [ ] 「写真から選択」→ フォトピッカー表示
- [ ] 写真選択 → 診断開始 → 結果表示

### 診断結果表示
- [ ] 総合スコア（円形プログレス）が正しい色で表示
- [ ] 4項目の詳細スコア（色褪せ・汚れ・ふわふわ感・ほつれ）
- [ ] コメントと推奨アクションのテキスト表示
- [ ] 写真サムネイルの表示（AsyncImage）

### 1日2回制限
- [ ] 1回目の診断後、「今日の診断: あと1回」と表示
- [ ] 2回目の診断後、ボタンが無効化（「あと0回」）
- [ ] 日付が変わると制限リセット

### 診断履歴
- [ ] 診断履歴一覧の表示（日付・スコア・推奨テキスト）
- [ ] タップ → 診断結果詳細画面に遷移
- [ ] スワイプ削除が動作する（写真も Storage から削除）
- [ ] 0件時「診断履歴がありません」表示

## 6. 家族グループ共有

### グループ作成
- [ ] 設定 → グループセクション →「タオルグループを作成」
- [ ] グループ名入力 → 作成成功
- [ ] 招待コード（6文字）が表示される
- [ ] 個人タオルがグループに移行される
- [ ] 一覧がグループのタオルを表示する

### グループ参加
- [ ] 「招待コードで参加」→ コード入力画面
- [ ] 有効なコード入力 → 参加成功
- [ ] 無効なコード → エラーメッセージ
- [ ] 参加後、グループのタオルが表示される

### グループ管理
- [ ] メンバー一覧の表示（オーナー・メンバー区別）
- [ ] 招待コードのコピー
- [ ] 「招待コードを再生成」→ 確認ダイアログ → 新コード生成
- [ ] 「グループから退出」→ 確認ダイアログ → 退出成功
- [ ] 退出後、個人モードに戻る（グループタオルは非表示）

### 最後のメンバーの退出
- [ ] 最後の1人が退出 → グループ全体が削除される

## 7. 設定画面

### アカウント
- [ ] 表示名の変更と保存
- [ ] リストアコードの表示とコピー
- [ ] サインアウト → 確認ダイアログ → サインイン画面に戻る

### 通知
- [ ] 「リマインダー通知」トグル ON → 権限リクエスト
- [ ] 通知時刻の変更
- [ ] 「期限切れリマインド」トグルの動作
- [ ] 通知 OFF → 通知がキャンセルされる

### Alexa 連携
- [ ] 「連携コードを生成」→ 6文字コード表示
- [ ] 有効期限（10分）の表示
- [ ] コード再生成が動作する

### アプリ情報
- [ ] バージョン番号が表示される
- [ ] プライバシーポリシーリンク → Safari でページが開く
- [ ] 利用規約リンク → Safari でページが開く

### アカウント削除
- [ ] 「アカウント削除」→ 確認ダイアログ（「この操作は取り消せません」）
- [ ] 確認 → 全データ削除（Firestore + Storage + Auth + Keychain）
- [ ] 削除完了 → サインイン画面に戻る
- [ ] グループに所属中の場合、グループからも退出される

- [ ] NavigationStack の push/pop が正常

## 9. エッジケース・異常系

### ネットワーク
- [ ] 機内モードで起動 → 適切なエラー表示（サインイン不可）
- [ ] 機内モードでタオル操作 → エラー表示
- [ ] ネットワーク復旧後に正常動作に戻る

### データ
- [ ] タオル0件の状態でアプリを使用
- [ ] タオル多数（10件以上）でのスクロール・パフォーマンス
- [ ] 長い名前・場所のテキスト表示（レイアウト崩れなし）
- [ ] 特殊文字（絵文字等）を名前に使用

### 認証
- [ ] サインアウト → 再度コード入力でサインイン
- [ ] アカウント削除 → 新規コードで再登録

## 10. App Store 審査要件

- [x] プライバシーポリシー URL がアクセス可能（`https://kaetao-c43f1.web.app/privacy-policy`）
- [x] 利用規約 URL がアクセス可能（`https://kaetao-c43f1.web.app/terms-of-use`）
- [x] `PrivacyInfo.xcprivacy` が存在し正しい内容
- [x] `ITSAppUsesNonExemptEncryption = false` が設定済み
- [x] カメラ使用目的（NSCameraUsageDescription）が適切
- [ ] アカウント削除機能が動作する（Apple 必須要件）
- [x] SettingsView にプライバシーポリシー・利用規約リンクあり
